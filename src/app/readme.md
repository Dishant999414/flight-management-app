# Flight Management App ✈️

A modern Flight Booking and Management System built using Next.js, Supabase, Zustand, Tailwind CSS, and PWA support.

---

# Features

- User Signup & Login Authentication
- Flight Search
- Dynamic Flight Listings
- Real-time Seat Selection
- Seat Reservation System
- Booking Confirmation with PNR
- My Bookings Page
- Progressive Web App (PWA)
- Responsive UI
- Realtime Seat Updates using Supabase

---

# Tech Stack

## Frontend
- Next.js 16
- React 19
- Tailwind CSS
- Zustand

## Backend
- Supabase
  - Authentication
  - PostgreSQL Database
  - Realtime
  - RPC Functions

## Other Libraries
- next-pwa
- react-hook-form
- zod
- uuid
